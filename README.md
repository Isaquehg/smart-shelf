# Smart-Shelf
<img src="Images\Logo_nome.png" alt="smart_shelf" width="300"/>

## About

The project is divided into 4 parts:
- Shelf and Scissor Actuator
- Shelf App
- Client App
- Cart

## Items/Tools Used

- Shelf:
    - ESP32
    - Ultrassonic Sensor HC-SR04
    - 60x10cm Wood
    - 128x64 OLED Display
    - I2C Modules
- Actuator:
    - Servo Motor MG996
    - 20cm Screw
    - 3D stems and suports
- Shelf App:
    - Android Studio
    - Java
    - SQL
    - MQTT Broker
- Client App:
    - Android Studio
    - Java
    - SQL
    - MQTT Broker
- Cart:
    - ESP32
    - Ultrassonic Sensor HC-SR04
